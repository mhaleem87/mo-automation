GIT COMMIT MESSAGE FORMAT:

git commit -m "docs: [TYPE] - [DESCRIPTION]"

Types:
- add      = New component/agent/bot
- update   = Modified existing component
- fix      = Bug fix in documentation
- integration = New integration added
- enhancement = New feature/capability

Examples:
- docs: add - New resume-screening-agent
- docs: update - Modify job search pipeline
- docs: fix - Correct API endpoint URL
- docs: integration - Add Slack webhook
- docs: enhancement - Add NLP support to bot
